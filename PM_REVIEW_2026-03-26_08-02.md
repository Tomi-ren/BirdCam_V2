## 3. 技术决策

### 架构设计

- **状态机**: 8状态FSM (S1音频监听 → S2广域搜索 → S3目标确认 → S4追踪准备 → S5目标追踪 → S6图像捕获 → S7释放 → S8回归)
- **事件驱动**: EventBus解耦各模块
- **搜索策略**: YOLO + 鸟类提示加权融合

### 关键配置

| 参数 | 值 | 说明 |
|------|-----|------|
| TARGET_READY_TH | 0.46 | 升级阈值 |
| VERIFY_PASS_TH | 0.68 | 验证通过阈值 |
| YOLO_CONF | 0.25 | 置信度 |
| TRACKER_TYPE | csrt | 追踪器类型 |
